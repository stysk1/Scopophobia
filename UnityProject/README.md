

# Unity Project for Scopophobia
** Scopophobia's Unity AssetBundle (Recreation)**

- This is my best guess at recreating the original AssetBundle for Scopophobia, from an AssetBundle Dump
- Full Compilation, working with the latest version of Scopophobia by TheUnknownCod3r
- To Compile, you will need to Setup the LethalCompanyUnityTemplate by EvaisaDev Linked below, and follow the setup guide. Then extract the Unity Project over the Template Folder provided by EvaisaDev. Also Build a new Scopophobia.dll, and copy it to Assets/Plugins, and rebuild your assetbundle before testing.
- Support is not given for this project, you should have at least a basic understanding of Unity Editor before making major changes to this.
- Any Internal changes should be thoroughly tested against the latest Scopophobia.dll built from Source, to ensure full compatibility.
